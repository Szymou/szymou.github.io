##如何push？

`git add xxx

git commit –m “sdsdad”//这时候仅仅是提交到本地仓库

git remote add origin git@github.com:Szymou/myblog.git//添加源仓库

如果返回remote origin already exists.

则先删除git remote rm origin

再添加git remote add origin git@github.com:Szymou/myblog.git//添加源仓库

然后将本地库上传到远程库git push origin master

`
